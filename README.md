# Resume Score Analyzer

A frontend application built using React.js that allows users to upload a resume (PDF), generates a simulated score, and provides improvement suggestions through a clean and modern UI.

---

## Features

- Upload resume in PDF format
- File type validation (PDF only)
- Simulated resume analysis (mocked logic)
- Score generation between 0–100
- Visual progress bar indicator
- Suggestions based on score
- Clean dark-themed user interface

---

## Tech Stack

- React.js
- JavaScript (ES6)
- HTML (JSX)
- CSS (Inline styling)

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/resume-analyzer.git

2. Navigate to the project folder:
   cd resume-analyzer

3. Install dependencies:
   npm install

4. Start the development server:
   npm start
